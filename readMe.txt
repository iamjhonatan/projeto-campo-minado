Projeto Campo Minado do curso de Java da Cod3r, ministrado pelo professor Leonardo Leitão.

O projeto visa fixar o conteúdo que foi apresentado até o momento, como;

* Orientação a objetos (encapsulamento, herança, polimorfismo e abstração);
* Stream API e suas funções;
* Funções lambda;
* Collections;
* Tratamento de erros;
* Além de fundamentos, estruturas de controle e outras bases da linguagem;

*Além de testes unitários na classe de Campo, utilizando o JUnit 5, que também constam no projeto.
